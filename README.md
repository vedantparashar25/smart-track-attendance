# SMART TRACK – Attendance Management System (CLI)
A lightweight and efficient **Command-Line Attendance Management System** built using Python.
Designed for quick student registration, attendance tracking, and automated attendance report
generation.
## FEATURES
• Add Students
• View Registered Students
• Mark Attendance (Present/Absent)
• JSON-based Data Storage
• Automatically Generated Attendance Reports (with percentages)
• Delete Students
## PROJECT STRUCTURE
attandance.py - Main Python script
attendance_data.json - Auto-generated JSON storage file
README.md - Project documentation
## HOW TO RUN
1. Clone the repository
2. Navigate to the project folder
3. Run: python attandance.py
## MAIN MENU OPTIONS
1. Add Student
2. View Students
3. Mark Attendance
4. View Attendance Report
5. Delete Student
6. Exit
